二次函数知识点

二次函数的一般形式为：f(x) = ax² + bx + c，其中a、b、c为常数，且a ≠ 0。

关键特征：

  • 开口方向：由a的正负决定，a>0时开口向上，a<0时开口向下
  • 顶点：抛物线的最高点或最低点，坐标为(-b/2a, (4ac-b²)/4a)
  • 对称轴：通过顶点且垂直于x轴的直线，方程为x = -b/2a
  • 与坐标轴交点：与y轴交于(0, c)，与x轴交点由判别式Δ=b²-4ac决定
  • 与直线y=t的交点：解方程ax²+bx+c=t，即ax²+bx+(c-t)=0

函数特征

顶点坐标: (0, 0)
x = -b/(2a) = 0, y = (4ac-b²)/(4a) = 0
对称轴: x = 0
x = -b/2a = 0
开口方向: 向上
与y轴交点: (0, 0)
与x轴交点: (0, 0)
与y=t交点: 无交点
x=s时y值: f(0) = 0
